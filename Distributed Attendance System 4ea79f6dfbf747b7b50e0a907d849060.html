Distributed Attendance System

How to download and use the app? Please refer to the sections below.

iOS Users:

Step1:

Download the TestFlight app from the AppStore using the following URL: https://apps.apple.com/us/app/testflight/id899247664

Step2:

Download the Distributed Attendance System BETA application from the following URL:

https://testflight.apple.com/join/S9Fe8yt6

Step3:

Install the Distributed Attendance System application and Register yourself using your VIT email, Register number, Name and password. Once you have registered you can verify your email through your Gmail. The verification mail can go to the spam so be sure to check there.

Step4:

Login using the credentials created - namely register number and password.

Step5:

Enjoy using the D.A.S Application and make sure you give us BETA feedback and screenshots of bugs if exists. You can send bugs by screenshotting within the application and pressing the share button on top right near the trash icon. In the options click "Share Beta Feedback" and write your review and submit.
